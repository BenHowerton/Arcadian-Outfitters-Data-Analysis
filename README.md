# Arcadian Outfitters Customer Sales Volume Dashboard

Interactive Vite + React dashboard that groups customers by average monthly sales volume to prioritize visit frequency.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
npm run preview
```

## GitHub Pages

This project is configured with `base: /Arcadian-Outfitters-Data-Analysis/` in `vite.config.js` and includes a GitHub Actions workflow that auto-deploys on every push to `main`.
